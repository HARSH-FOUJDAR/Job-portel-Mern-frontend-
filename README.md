

 MERN Stack Job Portal
 
LiveUrl = https://earnest-semifreddo-9217f3.netlify.app/

full-stack job portal application  Using MERN stack  and  Candidates  jobs apply the aor Recruiters  jobs post/manage the job 

 Key Features
 Candidate Portal
User Authentication: JWT-based secure login/signup.

Profile Management: Resume upload, skills integration, aur experience details.

Smart Search: Jobs ko title, location, aur category ke basis par filter karne ki suvidha.

Application Tracking: Apne apply kiye huye jobs ka live status dekhna.


 Tech Stack
Frontend: React.js, Tailwind CSS, Redux Toolkit (State Management)

Backend: Node.js, Express.js

Database: MongoDB Atlas

Authentication: JSON Web Tokens (JWT) & Bcrypt.js

File Storage: Cloudinary (Resumes aur Profile Pictures )
